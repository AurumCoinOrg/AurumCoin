# Aurum Positioning (Payments-Focused)

## Goal
Aurum is positioned as a payments-first PoW chain:
- fast confirmations
- simple on-chain transfers
- conservative, auditable engineering

## What we will NOT do
- premine / founder allocation
- stealth allocations
- confusing tokenomics

## What we WILL do
- ship a stable mainnet + wallets
- document parameters clearly
- publish a clean runbook + release checklist
- provide bootstrap/seed infrastructure
