# Aurum Exchange Integration Guide

This document is intended for exchanges and custodians integrating Aurum.

## Network
- Chain: **Aurum mainnet**
- Consensus: **Proof-of-Work (Scrypt)**

## Ports
- P2P: **19444**
- RPC: **19443**

## RPC
- Bitcoin-compatible JSON-RPC
- Authentication required (rpcuser / rpcpassword)

## Confirmations
- Deposits: **12 confirmations** (recommended)
- Withdrawals: **12 confirmations**

## Block Details
- Block time: **150 seconds**
- Difficulty retarget: **302,400 seconds (3.5 days)**

## Monetary Policy
- Initial subsidy: **56 AUR**
- Halving: **840,000 blocks**
- Max supply: **94,080,000 AUR**

## Notes
- Aurum is NOT a token
- Aurum runs on its own Layer-1 blockchain
- No premine or ICO
