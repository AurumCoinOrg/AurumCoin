# Aurum Launch Notes (Must-Complete)

## Before public testnet
- deterministic builds documented
- runbook + backup rules complete
- seed nodes online and tested
- explorer running

## Before mainnet public launch
- release tag created (versioned)
- binaries reproducible (or at least build steps verified)
- checkpoint strategy defined
- DNS seeds strategy finalized
- bootstrap method documented

## Before exchange outreach
- one-page spec sheet (params, ports, supply schedule)
- explorer endpoints
- contact + security policy
- signed release artifacts (recommended)
